# portfolio

My personal portfolio website — a retro terminal × neo-brutalist design built with pure HTML, CSS & JavaScript.

## ✨ Features

- 🖥️ Terminal boot sequence intro animation
- 🎨 Neo-brutalist design with bold borders and hard shadows
- 🌙 Dark/Light mode with smooth transitions
- ⚡ Glitch text effects and typewriter animations
- 📱 Fully responsive (mobile-first)
- 🚀 Zero dependencies — just open `index.html`

## 🛠️ Tech Stack

- HTML5
- Vanilla CSS (custom properties, keyframe animations)
- Vanilla JavaScript (Intersection Observer, theme toggle)
- Google Fonts (Space Grotesk, Inter, JetBrains Mono)

## 📁 Structure

```
portfolio/
├── index.html
├── css/
│   ├── style.css
│   ├── animations.css
│   └── themes.css
├── js/
│   ├── main.js
│   └── effects.js

    └── img/
```

## 🚀 Getting Started

Just open `index.html` in your browser. No build tools needed.
